KODI Popcorn Time

What it is

With KODI Popcorn Time you can search for movies that you can see immediately in KODI.

Download

Check out the Releases folder to download repository for installing and autoupdate add-on or you can download add-on separately.

Supported Platforms

  • XBMC/KODI 13.x and later
  • Windows x32 and x64
  • OS X x32 and x64
  • Linux x32, x64 and ARM
  • Raspberry Pi
  • Android ARM 4.0+
